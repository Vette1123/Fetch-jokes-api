# Joke Generator

Click the button, get a dad joke. Fetches from the [icanhazdadjoke](https://icanhazdadjoke.com) API with `async/await`.

[**Live demo →**](https://vette1123.github.io/Fetch-jokes-api/)

## Tech

- HTML
- CSS
- JavaScript (Fetch API, async/await)

## Run it locally

```bash
git clone https://github.com/Vette1123/Fetch-jokes-api.git
cd Fetch-jokes-api
# open index.html
```
